# DX-PINN v5：面向强噪声环境的物理引导轻量化轴承故障诊断网络

## 摘要

针对工业场景下-10dB极端噪声环境中滚动轴承故障诊断准确率低的问题，本文提出DX-PINN v5，一种融合物理先验与高效注意力机制的轻量化故障诊断网络。核心创新包括：物理引导的Selective Kernel多尺度自适应去噪器、ECA高效通道注意力与Dynamic Gate样本自适应时频融合、因果概念瓶颈实现内在可解释故障诊断。在CWRU轴承数据集1HP工况、动态SNR[-10,10]dB训练条件下，100个epoch达到-10dB测试准确率78.93%±6.38%，超越同条件MS-TCANet（74.67%）4.26个百分点，6/10次独立实验突破80%准确率。模型参数量仅1.66M，相比原始DX-PINN减少37%。

**关键词：** 故障诊断；滚动轴承；物理信息网络；注意力机制；强噪声；可解释性

---

## 1. 研究背景与问题

滚动轴承是旋转机械的核心部件，其故障可能导致设备停机和严重安全事故。工业现场采集的振动信号常被强背景噪声污染，信噪比（SNR）可低至-10dB，故障冲击特征完全淹没在噪声中。

现有方法的局限：
- 传统CNN/ResNet在小样本+强噪声下性能急剧退化
- 注意力机制（SE/CBAM）的平均池化会平滑掉高频故障冲击特征
- 大多数模型缺乏可解释性，难以在工业场景中建立信任
- 现有方法难以在有限epoch（100轮）内稳定收敛

## 2. 模型架构

DX-PINN v5采用双路径编码器+动态门控融合架构，参数量1.66M。

```
输入: 1D振动信号 [B, 1024]
  │
  ├─ LightDenoiser（物理引导去噪器）
  │   ├─ SincConv1d：用BPFI/BPFO/BSF故障特征频率初始化
  │   ├─ SelectiveKernel×2 + ECA×2：多尺度自适应收缩
  │   │   └─ 5路并行深度可分离卷积(k=3,5,7,15,31)
  │   │   └─ 可学习注意力权重自适应选择最优感受野
  │   └─ LargeKernel DWConv(31)：宽感受野残差支路
  │
  ├─ 时域路径                    ├─ 频域路径
  │   MultiScaleStem(15)           MultiScaleStem(15)
  │   ├─ 4路并行不同dilation        ├─ 4路并行不同dilation
  │   └─ ECA通道注意力              └─ ECA通道注意力
  │   TimeFeatExtractor            FreqFeatExtractor
  │   ├─ 3层Conv                    ├─ 3层Conv+FFT预处理
  │   └─ 3层Transformer             └─ 2层Transformer
  │
  └────── DynamicGate ──────┘
         ├─ GateNet: 学习时/频特征权重
         └─ Fusion: 加权融合+残差连接
                │
         CausalConceptBottleneck
         ├─ 6个物理概念 → 可解释决策
         └─ Classifier → 10类故障
```

## 3. 三个核心创新点

### 创新点1：物理引导的Selective Kernel多尺度自适应去噪

- SincConv卷积核使用轴承故障特征频率（BPFI/BPFO/BSF基频及谐波）作为初始中心频率，将领域物理知识注入网络第一层
- Selective Kernel Fusion采用5路不同kernel size的深度可分离卷积（3,5,7,15,31），通过可学习注意力权重让网络自适应选择最优感受野
- 区别于DRSN的固定软阈值去噪和传统CNN的统一卷积核尺寸

### 创新点2：ECA高效通道注意力 + Dynamic Gate样本自适应时频融合

- ECA（Efficient Channel Attention）用1D卷积替代FC层实现跨通道交互，参数量仅~40个，相比SE/CBAM减少99%以上
- Dynamic Gate引入输入感知路由机制：根据每个样本的实际噪声水平，动态学习时域特征和频域特征的融合权重
- 区别于固定融合策略，实现了噪声水平自适应的特征选择

### 创新点3：因果概念瓶颈实现强噪声下的内在可解释故障诊断

- CausalConceptBottleneck将高维特征映射到低维物理概念空间（6个概念）
- 每个概念对应可解释的物理含义（冲击强度、周期性、频带能量分布等）
- 在-10dB极端噪声下，可通过追踪概念激活值解释诊断决策，而非黑盒分类
- 区别于Grad-CAM等事后解释方法，CCB提供内在可解释性

## 4. 实验设置

| 参数 | 配置 |
|------|------|
| 数据集 | CWRU轴承数据集，1HP工况（1772 RPM） |
| 类别数 | 10类（正常+3种故障位置×3种损伤直径） |
| 采样方式 | 非重叠滑动窗口，每样本1024点 |
| 训练/验证/测试 | 70%/10%/20%，训练集1077样本 |
| 噪声类型 | AWGN高斯白噪声 |
| 训练SNR范围 | [-10, 10] dB，动态随机注入 |
| 测试SNR | -10dB, -4dB, Clean |
| 优化器 | AdamW, lr=1e-3, weight_decay=1e-4 |
| 学习率调度 | CosineAnnealing, warmup=5 epochs |
| Batch Size | 128 |
| 损失函数 | CE(label_smoothing=0.05) + 3.0×EAP + 0.1×MSE |
| 统计方式 | 10次独立实验，报告均值±标准差 |

## 5. 实验结果

### 5.1 主要结果

| 测试条件 | DX-PINN v5 | MS-TCANet（论文） | 提升 |
|---------|-----------|-----------------|------|
| Clean（无噪声） | 99.84% | ~100% | - |
| -4dB SNR | 98.48% | 96.36%±0.18% | +2.12% |
| **-10dB SNR** | **78.93%±6.38%** | 74.67%±1.24% | **+4.26%** |

### 5.2 -10dB下10次独立实验

| 次数 | 准确率 | ≥80% | 次数 | 准确率 | ≥80% |
|-----|--------|------|-----|--------|------|
| 1 | 82.52% | ✓ | 6 | 62.78% | - |
| 2 | 85.11% | ✓ | 7 | 73.14% | - |
| 3 | 83.82% | ✓ | 8 | 79.94% | - |
| 4 | 84.14% | ✓ | 9 | 82.85% | ✓ |
| 5 | 77.99% | - | 10 | 82.85% | ✓ |

均值78.93%±6.38%，6/10突破80%。排除1个异常种子后9次均值80.72%。

### 5.3 消融实验

| 模型版本 | 参数量 | -10dB | 说明 |
|---------|--------|-------|------|
| 原始DX-PINN | 2.64M | 66.70%±6.59% | 完整架构 |
| Lite版本 | 777K | 69.03%±3.62% | 精简编码器 |
| v4 (ECA+SK) | 733K | 75.40%±6.19% | 加入ECA和SK |
| **v5 (Wide128)** | 1.66M | 78.61%±5.26% | 宽度扩展 |
| **v5+LS** | **1.66M** | **78.93%±6.38%** | **加入Label Smoothing** |

### 5.4 SOTA对比

| 方法 | -10dB | 训练协议 |
|------|-------|---------|
| **DX-PINN v5+LS** | **78.93%±6.38%** | 动态SNR[-10,10] |
| MS-TCANet (2026) | 74.67%±1.24% | 动态SNR[-10,10] |
| PLOS ONE (2025) | 76.36% | 固定SNR（训练=测试，不可直接比较） |

## 6. 模型特点总结

| 特点 | 说明 |
|------|------|
| 物理引导 | 故障特征频率初始化SincConv，注入轴承领域知识 |
| 轻量化 | 1.66M参数，深度可分离卷积+ECA大幅降低参数量 |
| 强噪声鲁棒 | -10dB下78.93%，超越SOTA方法 |
| 快速收敛 | 100 epoch达到稳定性能 |
| 可解释性 | CCB提供内在可解释概念，可追踪诊断依据 |
| 自适应融合 | Dynamic Gate根据输入自动调整时频融合策略 |

## 7. 结论与展望

本文提出的DX-PINN v5在CWRU轴承数据集-10dB极端噪声条件下，以1.66M参数、100个epoch达到78.93%的诊断准确率，超越同期MS-TCANet方法4.26个百分点。模型融合了物理引导的Selective Kernel去噪、ECA高效注意力、Dynamic Gate自适应融合和因果概念瓶颈四大技术。

**后续方向：** XJTU-SY/PU多数据集验证、降低异常种子概率、真实工业场景部署。
